addEventListener(type, listener, useCapture)

Registers a listener on the specified event. The possible event types are:
  • 'error' (see error event specification)
  • 'warning' (see warning event specification)
  • 'manifestUrlUpdate' (see manifestUrlUpdate event specification)
  • 'play_bitrate' (see play_bitrate event specification)
  • 'download_bitrate' (see download_bitrate event specification)
  • 'bufferLevel_updated' (see bufferLevel_updated event specification)
  • 'state_changed' (see state_changed event specification)
  • 'cueEnter' (see cueEnter event specification)
  • 'cueExit' (see cueExit event specification)
    • Parameters:
    Name Type Description
    type string the event type for listen to, either any HTML video element event or player event.
    listener callback the callback which is called when an event of the specified type occurs
    useCapture boolean see HTML DOM addEventListener() method specification

    Type Definitions

    PlayerParams

    Player parameters object. All parameters values are applied for any stream type. Parameters can be overriden specifically for audio and video track by setting parameters values in the params.audio and params.video objects.
    Type:
    • Object
    Properties:
    Name Type Description
    BufferController.minBufferTimeForPlaying number Minimum buffer level before playing, in seconds (default value = 0)
    BufferController.minBufferTime number Minimum buffer size (in seconds), if set to '-1' the maximum value between the manifest's minBufferTime and 16 sec. is considered (default value = -1)
    BufferController.bufferToKeep number The buffer size (in seconds) to keep anterior to current playing time (default value = 30)
    BufferController.liveDelay number The delay (in seconds) between the live edge and playing time, if set to '-1' the live delay is set according to minBufferTime (default value = -1)
    ABR.minBandwidth number Minimum bandwidth to be playbacked (default value = -1)
    ABR.maxBandwidth number Maximum bandwidth to be playbacked (default value = -1)
    ABR.minQuality number Minimum quality index (start from 0) to be playbacked (default value = -1)
    ABR.maxQuality number Maximum quality index (start from 0) to be playbacked (default value = -1)
    ABR.switchUpIncrementally boolean Switch up quality incrementally, or not (default value = false)
    ABR.switchUpRatioSafetyFactor number Switch up bandwith ratio safety factor (default value = 1.5)
    ABR.latencyInBandwidth boolean Include (or not) latency in bandwidth (default value = true)
    ABR.switchLowerBufferTime number Buffer level (in seconds) under which switching down to lowest quality occurs (default value = -1)
    ABR.switchLowerBufferRatio number Buffer level (as percentage of buffer size) under which switching down to lowest quality occurs (default value = 0.25)
    ABR.switchDownBufferTime number Buffer level (in seconds) under which switching down quality occur, if unsufficient bandwidth (default value = -1)
    ABR.switchDownBufferRatio number Buffer level (as percentage of buffer size) under which switching down quality occurs, if unsufficient bandwidth (default value = 0.5)
    ABR.switchUpBufferTime number Buffer level (in seconds) upper which switching up quality occurs, if sufficient bandwidth (default value = -1)
    ABR.switchUpBufferRatio number Buffer level (as percentage of buffer size) upper which switching up quality occurs, if sufficient bandwidth (default value = 0.75)
    ABR.droppedFramesMinRatio number The number of dropped frames (as a ratio to total video frames) from which switching up quality is disabled (default value = 0.1)
    ABR.droppedFramesMaxRatio number The number of dropped frames (as a ratio to total video frames) from which quality is switched down (default value = 0.3)
    ManifestLoader.RetryAttempts number Number of retry attempts for downloading manifest file when it fails (default value = 2)
    ManifestLoader.RetryInterval number Interval (in milliseconds) between each retry attempts for downloading manifest file (default value = 500)
    FragmentLoader.RetryAttempts number Number of retry attempts for downloading segment files when it fails (default value = 2)
    FragmentLoader.RetryInterval number Interval (in milliseconds) between each retry attempts for downloading segment files (default value = 500)
    Protection.licensePersistence boolean Provides or not license persistence at application level, in case no persistence is provided by the CDM (default value = false)
    video Object Video parameters (parameters for video track)
    audio Object audio parameters (parameters for audio track)
